04

Three-Theme with System Option

When using a System button, check data-theme attribute (user's choice) for active states.

data-theme (user's choice)

Effective theme (class)

11

Programmatic API

Control themes via JavaScript API.

ThemeManager.current (effective)
—
ThemeManager.theme (user's choice)
—
ThemeManager.themes
—
ThemeManager.setTheme('dark'); // user chose dark ThemeManager.setTheme('system'); // user chose system ThemeManager.current; // effective theme ("light" or "dark") ThemeManager.theme; // user's choice ("system", "dark", etc.)
12

Event Handling

Listen to 'themechange' events for custom logic.

Waiting for theme changes...
window.addEventListener('themechange', e => { e.detail.theme; // effective theme e.detail.userTheme; // user's choice (can be "system") e.detail.effectiveChanged;// did visual appearance change? });
