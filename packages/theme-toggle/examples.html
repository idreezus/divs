Theme Toggle

A collection of minimal, elegant theme toggle patterns. Pure CSS active states with declarative data attributes.

Current: —

Example 01

Simple Text Toggle

Clean text buttons in a pill container. Active state styled via CSS class on <html>.

View Code 
<div class="toggle-1">
  <button class="btn-light" data-theme-value="light">Light</button>
  <button class="btn-dark" data-theme-value="dark">Dark</button>
</div>

/* Active states via CSS class on <html> */
.light .toggle-1 .btn-light,
.dark .toggle-1 .btn-dark {
  background: var(--bg);
  color: var(--text);
  box-shadow: 0 1px 3px var(--shadow);
}

Example 15

Select Dropdown

Native <select> element. Requires minimal JS to sync with ThemeManager.

View Code 
<select class="toggle-15" id="theme-select">
  <option value="light">Light</option>
  <option value="dark">Dark</option>
  <option value="system">System</option>
</select>

// Sync select with ThemeManager
const select = document.getElementById('theme-select');
select.addEventListener('change', e => ThemeManager.setTheme(e.target.value));

// Update select when theme changes
window.addEventListener('themechange', e => {
  select.value = e.detail.userTheme;
});
Example 16

Programmatic API

Control themes via JavaScript API. Useful for integrating with other systems.

ThemeManager.current
—
ThemeManager.theme
—
ThemeManager.themes
—
View Code 
// Set theme explicitly
ThemeManager.setTheme('dark');

// Set to follow system preference
ThemeManager.setTheme('system');

// Get current effective theme (never "system")
ThemeManager.current; // "dark"

// Get user's choice (can be "system")
ThemeManager.theme; // "system"

// Cycle to next theme (skips "system")
ThemeManager.toggleTheme();

// Available themes
ThemeManager.themes; // ["light", "dark", ...]
Example 17

Event Handling

Listen to 'themechange' events for custom logic and animations.

Waiting for theme changes...
View Code 
window.addEventListener('themechange', e => {
  console.log('Effective:', e.detail.theme);      // "dark"
  console.log('Previous:', e.detail.previous);    // "light"
  console.log('User chose:', e.detail.userTheme); // "system"
  console.log('Changed:', e.detail.effectiveChanged); // true
});

// Event detail properties:
// - theme: current effective theme (never "system")
// - previous: previous effective theme
// - userTheme: user's choice (can be "system")
// - previousUserTheme: previous user's choice
// - effectiveChanged: did visual appearance change?
