Basic Functionality

Simple horizontal tabs, vertical orientation, and default tab selection

01

Basic Horizontal Tabs

Simple horizontal tabs with default configuration. Click triggers to switch between panels.

Overview - This is the overview panel content. It provides a high-level summary of the product or service being described.
Features - Explore all the amazing features including automatic initialization, keyboard navigation, ARIA support, and URL synchronization.
Pricing - Our flexible pricing plans are designed to fit teams of all sizes. Start free and scale as you grow.
02

Vertical Orientation

Tabs with vertical orientation. Arrow Up/Down keys navigate instead of Left/Right.

Account Settings - Manage your profile information, update your email address, and customize your display name.
Security Settings - Configure two-factor authentication, manage active sessions, and update your password.
Notification Preferences - Control which emails you receive and how you're notified about important updates.
Billing Information - View invoices, update payment methods, and manage your subscription plan.
Use Arrow Up/Down to navigate. Arrow Left/Right are ignored in vertical mode.
03

Default Tab Selection

Tabs with a specific default tab set via data-tabs-default. The "Advanced" tab should be active on load.

Basic Mode - Simple configuration with sensible defaults. Perfect for getting started quickly.
Advanced Mode - This tab is selected by default using data-tabs-default="advanced". More options for power users.
Expert Mode - Full control over every setting. Recommended for experienced users only.
